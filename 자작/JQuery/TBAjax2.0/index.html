TBAJAX 통신값 변수로 받기 소스

<script type="text/javascript"> var wow = $.TBAjax('./content/page.php', '?test=thk', 'true'); alert(wow); </script>

TBAJAX 옵션 Attribute

- tb-ajax-target의 객체 타입에 따라 출력 값이 다릅니다.
tb-ajax-action TBAjax 가 동작 하기위한 액션을 설정 합니다. (click, submit, load)
tb-ajax-target 결과출력 할 객체를 설정 합니다.
tb-ajax-url ajax 통신할 주소를 설정 합니다. * form일경우 제외
tb-ajax-data 통신할 파라미터를 설정 합니다. (예: ?name=홍길동) * form일경우 제외
tb-ajax-log 통신 로그 출력 여부를 설정 합니다.

tb-ajax-action="load" 소스

<div id="LoadHtml" tb-ajax-action="load" tb-ajax-target="#LoadHtml" tb-ajax-url="./ajax/page.php" tb-ajax-data="?test=wow" tb-ajax-log="">대기중</div>
